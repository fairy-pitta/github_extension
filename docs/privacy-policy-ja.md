# GitHub Dashboard プライバシーポリシー

**最終更新日:** 2025年1月13日

## はじめに

GitHub Dashboard（「当拡張機能」「私たち」）は、GitHub.com上でGitHubのアクティビティを表示するChrome拡張機能です。本プライバシーポリシーでは、当拡張機能がどのように情報を収集、使用、保護するかを説明します。

## データの収集

### アクセスする情報

当拡張機能は、GitHub APIを通じて以下の情報にアクセスします：

- **ユーザー情報**: GitHubユーザー名、表示名、アバターURL、自己紹介、所在地、会社、ウェブサイト、ソーシャルメディアリンク
- **リポジトリ情報**: リポジトリ名、説明、更新タイムスタンプ、公開状態（公開/非公開）、リポジトリ所有者
- **プルリクエストデータ**: プルリクエストのタイトル、番号、状態、URL、作成/更新日、レビュー状況、コメント、関連リポジトリ情報
- **イシューデータ**: イシューのタイトル、番号、状態、URL、更新日、ラベル、コメント、担当者、関連リポジトリ情報
- **コントリビューションデータ**: コントリビューションカレンダーデータ、コミット数、コントリビューションストリーク
- **アクティビティ統計**: 週次・月次の統計情報（コミット、プルリクエスト、レビュー、イシュー、コメント）

### ローカルに保存する情報

当拡張機能は、Chromeのローカルストレージを使用して、以下の情報をデバイスにローカル保存します：

- **認証トークン**: GitHub OAuthアクセストークンまたはPersonal Access Token（暗号化されて安全に保存）
- **キャッシュデータ**: パフォーマンス向上とAPI呼び出し削減のための一時的なAPIレスポンスのキャッシュ
- **ユーザー設定**: テーマ設定、言語設定、お気に入りリポジトリ、ダッシュボード表示設定

## データの使用方法

### 目的

収集したデータは、以下の目的のみに使用されます：

1. **ダッシュボードの表示**: GitHubアクティビティを統合ダッシュボードインターフェースで表示
2. **パフォーマンス最適化**: データをローカルにキャッシュし、GitHubへのAPI呼び出しを削減
3. **ユーザー体験**: ユーザーの設定と好みを記憶

### データ処理

- **読み取り専用操作**: 当拡張機能は読み取り専用操作のみを実行します。GitHubアカウントのデータを変更、作成、更新、削除することはありません
- **ローカル処理**: すべてのデータ処理はデバイス上でローカルに実行されます。GitHubの公式API以外の外部サーバーにデータを送信することはありません
- **第三者への共有なし**: データを第三者と共有、販売、送信することはありません

## データの保存

### ローカルストレージ

- すべてのデータは、Chromeの`chrome.storage.local` APIを使用してデバイスにローカル保存されます
- データはデバイスに残り、外部サーバーに送信されることはありません
- 拡張機能をアンインストールするか、Chromeの拡張機能ストレージをクリアすることで、いつでも保存されたデータを削除できます

### キャッシュ管理

- キャッシュデータには有効期限（TTL）があり、自動的に更新されます
- リフレッシュボタンを使用して、いつでも手動でデータを更新できます
- ログアウトまたは拡張機能をアンインストールすると、キャッシュはクリアされます

## データセキュリティ

### セキュリティ対策

- **暗号化されたストレージ**: 認証トークンは、Chromeの組み込み暗号化を使用して安全に保存されます
- **HTTPSのみ**: すべてのAPI通信はHTTPS暗号化を使用します
- **外部送信なし**: GitHubの公式API以外のサーバーにデータを送信することはありません
- **読み取り専用アクセス**: 当拡張機能は読み取り権限のみを要求し、GitHubデータを変更することはできません

### トークンのセキュリティ

- GitHubアクセストークンはローカルに保存され、Chromeによって暗号化されます
- トークンが第三者と共有されることはありません
- GitHubの設定からいつでもアクセスを取り消すことができます

## あなたの権利

データに関する以下の権利があります：

1. **アクセス**: Chromeの拡張機能ストレージビューアーを通じて、拡張機能によって保存されたすべてのデータを表示できます
2. **削除**: 拡張機能をアンインストールするか、Chromeの拡張機能ストレージをクリアすることで、保存されたすべてのデータを削除できます
3. **制御**: GitHubのOAuth設定から、いつでも拡張機能のGitHubアカウントへのアクセスを取り消すことができます
4. **透明性**: すべてのデータアクセスは、Chromeの拡張機能権限で確認できます

## 第三者サービス

### GitHub API

- 当拡張機能は、GitHubの公式API（api.github.com）と直接通信します
- すべてのAPIリクエストは、GitHubアクセストークンを使用して認証されます
- 中間サービスやプロキシを使用することはありません
- GitHubのプライバシーポリシーが、APIを通じてアクセスされるデータに適用されます: https://docs.github.com/ja/site-policy/privacy-policies/github-privacy-statement

## 児童のプライバシー

当拡張機能は13歳未満のユーザーを対象としていません。13歳未満の児童から個人情報を故意に収集することはありません。
